Key Invoice Features

  • A4 Constraint: All content fits on a single A4 page with responsive font scaling
  • Professional Layout: Three-section header with logo, title, and company address
  • Flexible Content: Supports up to 31 days of daily summary and 7 products
  • GST Compliance: Proper GST breakdown with subtotal and grand total
  • QR Code Integration: Payment QR code positioned strategically
  • Daily Summary: 4-column grid layout for transaction history
  • Print Optimization: Exact color reproduction and page break control
  • PureDairy Branding: Consistent branding with logo and footer contact information
